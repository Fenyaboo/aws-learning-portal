# Studies With AI (SWA)

Studies With AI is an AI-powered learning platform designed to help students generate exercises, summaries, and learning materials.

## Features
- AI generated exercises
- Learning material storage
- Smart study assistant

## Architecture
The platform is built on AWS cloud services:

- Amazon EC2 for compute
- Amazon S3 for storing documents
- CloudFront for content delivery
